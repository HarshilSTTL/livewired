### [2026-05-28 15:45] | FIX | `get_notifications` — Remove 2-day notification history limit

**Bug Fix:** Fixed unread notification badge showing count despite empty notification list. #notifications #bug-fix

**Root Cause:** Inconsistency between two APIs:
- [[get_unread_notification_count]] — counted ALL unread notifications (no time limit)
- [[get_notifications]] — showed ONLY notifications from past 2 days

**Example Scenario:**
1. User has 5 old unread notifications (older than 2 days)
2. Badge shows "5 unread" (from [[get_unread_notification_count]])
3. Notification page is empty (from [[get_notifications]] filtering by 2-day window)

**Fix Applied:**
- Removed `AND created_at >= NOW() - INTERVAL '2 days'` from [[get_notifications]] query
- Both APIs now consistent: show ALL notifications with no time limit
- Updated function comment and documentation

**Files Modified:**
1. [[functions/notifications/get_notifications.md]] — Removed 2-day time filter in SQL
2. [[docs/api/notifications/get_notifications.md]] — Updated documentation notes

**Related Functions:**
- [[get_unread_notification_count]] — Badge count (filters: is_read=false, is_cleared=false)
- [[mark_notifications_read]] — Mark notifications as read
- [[clear_notifications]] — Hide/clear notifications

**Impact:**
- ✅ Unread badge and notification list are now in sync
- ✅ Users see complete notification history
- ✅ No notifications hidden due to age
- ✅ Better data consistency across APIs

**Commit:** `84757a0` — fix: remove 2-day notification history limit

### [2026-05-28 15:30] | FIX | `search_profiles` — Filter platforms to only show main streaming platforms

**Enhancement:** Updated [[search_profiles]] API to exclude custom links and additional links from platform results. #search #platforms #enhancement

**Reason:** Search page should display only main streaming platforms. Custom links and additional links should only appear on the full profile detail page via [[get_profile_custom_links]].

**Change Made:**

**Platform IDs Whitelisted:**

**Related APIs:**

**Impact:**
- ✅ Search results show cleaner platform icons (4 main platforms only)

- ✅ No breaking changes; cleaner UX for profile discovery
