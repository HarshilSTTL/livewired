### [2026-04-14 00:00] | SP | manage_profile_platform — replace-aware SP for platform links

**What it does:**
Replace-aware SP for `creator_platform_accounts`. Same pattern as `manage_custom_links`:
- `id` present → UPDATE `channel_url`
- `id` null → INSERT new row (requires `platform_id`)
- Row in DB but not in list → soft-delete
- `p_platforms = []` → soft-delete all platform links for the profile

**Parameters:** `p_profile_id uuid`, `p_user_id uuid`, `p_platforms jsonb`

**Request shape:**
```json
{
  "p_profile_id": "profile-uuid",
  "p_user_id":    "user-uuid",
  "p_platforms": [
    { "id": "existing-uuid", "platform_id": null, "channel_url": "https://youtube.com/@handle" },
    { "id": null,            "platform_id": 2,    "channel_url": "https://twitch.tv/handle" }
  ]
}
```
