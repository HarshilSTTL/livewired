```sql

SELECT COALESCE(json_agg(...), '[]'::json) FROM (...)

-- Prevents scoping errors in multi-level correlated subqueries
```

**Why This Works:**
- Placing COALESCE inside the SELECT statement maintains proper table alias scoping
- The subquery (inner FROM clause) can reference `cp.id` from the outer scope
- json_agg(...) returns NULL when no rows exist, COALESCE handles default empty array
