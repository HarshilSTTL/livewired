### [2026-04-09 00:00] | SP | get_notifications — fetch user's past 2-day notification history

**What changed:**
New read SP that returns the authenticated user's notifications from the past 2 days, sorted latest first.

**Params:**
- `p_user_id uuid` — required; filters notifications to this user

**Logic:**
- Null guard on `p_user_id`
- Selects from `notifications` WHERE `user_id = p_user_id` AND `created_at >= NOW() - INTERVAL '2 days'`
- Orders by `created_at DESC`
- Returns `[]` if no notifications in window (never null)

**Response fields:** `id`, `title`, `body`, `data` (jsonb), `created_at`
