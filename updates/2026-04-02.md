### [2026-04-02 17:00] | SP | get_suggested_profiles — personalised creator suggestions for onboarding

**New SP:** `get_suggested_profiles` (POST /rpc/get_suggested_profiles)

Used after `submit_platform` + `submit_tags` in onboarding to show creators to follow.

**Scoring:** `match_score = platform_matches + tag_matches`
- Platform match: creator has a platform the user also selected (`user_preferred_platforms`)
- Tag match: creator has a tag the user also selected (`user_interests`)
- Results ordered: match_score DESC → followers DESC

**Exclusions:**
- Already-followed profiles (`follows.is_active = true`)
- User's own profiles (`creator_profiles.user_id = p_user_id`)

**Edge case:** User skipped platform/tag steps → all scores = 0 → ordered by followers DESC

**Files changed:**
- `functions/profiles/get_suggested_profiles.md` — NEW SP SQL
- `docs/api/profiles/get_suggested_profiles.md` — NEW full API docs
